# Adivinhador de Idade - Um teste em Svelte
Essa é uma aplicação simples que acessa uma api e retorna uma idade altamente precisa, baseada no nome digitado. 
Você pode se surpreender... ou não! 

## Pré Requisitos

- [Git](https://git-scm.com/downloads "Git")
- [Node.js 18+](https://nodejs.org/ "NodeJs")

### 1. Clone o projeto

```bash
  git clone git@github.com:JoaoGabrielVargas/sveltekit-test-app.git
  cd sveltekit-test-app
```

### 2. Configurar e executar o Svelte local

```bash
  npm install
  npm run dev
```
